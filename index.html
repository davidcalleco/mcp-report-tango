Asset Performance Audit
Scope: Full asset library (failed) → Newsletter CMS collection (13 items)

Full Asset Library Scan — MCP Limitation

The asset_tool timed out when attempting to enumerate the full asset library. The Webflow API lacks pagination and folder-scoping for large asset libraries. Folder structure was retrievable (30+ folders), but individual asset enumeration was not feasible at scale.

What MCP Cannot Do (Requires Hybrid Approach)

CapabilityLimitationWorkaround
SEO analysisNo parsing engineFetch staging HTML + parser
Accessibility analysisNo WCAG scanningFetch staging HTML + parser
Link validationNo crawler/HTTP checkerFetch HTML + HTTP requests
Full asset library scanTimeout on large librariesAudit via CMS collections
Custom embed elementsNot accessible via APIManual fix required
The Hybrid Approach: The most effective audit methodology combines MCP tools for structured data retrieval and automated fixes, staging URLs with HTML parsing for SEO and accessibility detection, and web search for enriching content with real-world data. This achieves both detection and remediation in a single workflow — a strong model for pre-publish QA automation.
